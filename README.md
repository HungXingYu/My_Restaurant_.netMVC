<h2 id="1"> 專案概述--Project Description</h2>

> 學習.net MVC 時練習用專案
>
> 功能目前僅有：
>
> 1.  分頁瀏覽餐廳清單
> 2.  以餐廳中、英文名進行模糊搜尋
> 3.  顯示各餐廳詳細資料
> 4.  新增一筆餐廳資料
>

<h2 id="2"> 開發工具與環境--Prerequisites</h2>

1. Windows 11
2. Visual Studio 2019
3. MS-SQL (需2012以上版本)

<h2 id="3"> 使用語言--Language</h2>

1. C#
2. Bootstrap
3. css
4. Javascript
5. jQuery
